### ⏳ EST : MÉMOIRE TEMPORELLE
![Ignis](https://img.shields.io/badge/IGNIS-e65100?style=for-the-badge&logo=flame&logoColor=white)

**Stratigraphie des décisions**

Les choix s'accumulent en couches géologiques : l'immédiat, l'historique, l'archétypal. Comprendre le présent exige la lecture de ces temporalités superposées.

- **Couche récente** : Décisions visibles et leurs répercussions directes
- **Couche sédimentaire** : Héritages institutionnels et culturels  
- **Couche fondatrice** : Structures profondes qui persistent

*La pluralité n'émerge qu'en intégrant cette profondeur temporelle.*
