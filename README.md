# Traité de proposition jardinière
## La collapsologie pour la préservation

---

### Partie 1 : Le tyran cyclope

Le système décisionnel monoculaire ignore les cycles naturels. L'autorité centralisée impose des protocoles uniformes sur des territoires diversifiés, méconnaissant les variations locales. Cette cécité structurelle produit des interventions inadéquates qui perturbent les équilibres.

L'effondrement résulte de cette inadéquation entre échelle de décision et échelle d'action.

---

### Partie 2 : Le symbole caché

Sous la rationalité gestionnaire se dissimule une logique extractive. Les indicateurs occultent les dynamiques de régénération, comptabilisant les prélèvements sans intégrer les temps de reconstitution.

La jardinière décode cette symbolique : la productivité réelle s'enracine dans la préservation des capacités régénératives.

---

### Partie 3 : La collaboration en strates

La solution s'organise selon une architecture stratifiée reproduisant les logiques écosystémiques :

**Strate primaire** : Parcelle - autorité sur les interventions immédiates  
**Strate secondaire** : Coordination - gestion des flux entre parcelles  
**Strate tertiaire** : Planification - orientations à moyen terme

Cette organisation évite la tyrannie cyclope en distribuant l'intelligence décisionnelle selon les compétences effectives. L'harmonie tellurique émerge de la correspondance entre organisation humaine et naturelle.
